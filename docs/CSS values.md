# CSS values

```css
margin: 10px 20px 30px 40px; /* the same as 1. margin-top 2. margin-right 3. margin-bottom 4. margin-left  */
margin: 10px 20px 40px; /* the same as 1. margin-top 2. margin-left and margin-right 3. margin-bottom  */
margin: 10px 20px; /* the same as 1. margin-top and margin-bottom 2. margin-left and margin-right */
margin: 10px; /* the same margin from all sides */
```
