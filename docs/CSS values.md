# CSS values

```css
margin: 1px 2px 3px 4px; /* the same as 1. margin-top 2. margin-right 3. margin-bottom 4. margin-left  */
margin: 1px 2px 4px; /* the same as 1. margin-top 2. margin-left and margin-right 3. margin-bottom  */
margin: 1px 2px; /* the same as 1. margin-top and margin-bottom 2. margin-left and margin-right */
```
